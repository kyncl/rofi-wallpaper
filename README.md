# rofi-wallpaper
using rofi to set new wallpaper


## Dependencies
### hard dependencies
rofi
### light dependencies (more in section tweaks)
pywal
swww

## Installation/Usage
```bash
git clone https://github.com/kyncl/rofi-wallpaper/
cd rofi-wallpaper
./rofi-wallpaper path/to/wallpapers (most likely ~/Pictures/wallpapers)
```
